Astro Arena

Instruction

  • Space explorer is a simple, fun, and strategic single-player game played in outer space.
  • Control the spaceship using arrow keys or touch screen (up, left, and right keys).
  • Destroy asteroids using the spaceship's weapon.
  • Asteroids will appear randomly on the screen; avoid collisions to keep the ship safe.
  • The game ends if the spaceship collides with an asteroid.
